# ☁️ Cloud Stock Analysis: MSFT, ORCL, AMZN  
**Python | Financial Analysis | Time Series | Data Visualization**

Built a financial analysis project to evaluate the performance of major cloud computing companies, helping identify the strongest long-term investment opportunity based on data-driven insights.

---

## 📊 Project Overview
![Microsoft Analysis](images/msft_chart.png)

This project analyzes 5 years of stock data (2020–2025) for:
- Microsoft (MSFT)  
- Oracle (ORCL)  
- Amazon (AMZN)  

Focused on understanding:
- Price trends over time  
- Market growth patterns  
- Comparative performance across cloud providers  

---

## 📈 Microsoft (MSFT) Analysis
![Microsoft](images/msft_chart.png)

- Strong and consistent upward trend  
- Balanced growth with lower volatility  
- Industry leader in cloud computing (Azure)  

---

## 📉 Oracle (ORCL) Analysis
![Oracle](images/orcl_chart.png)

- Steady, moderate growth over time  
- Lower volatility compared to competitors  
- Positioned as a stable, income-oriented investment  

---

## 🚀 Amazon (AMZN) Analysis
![Amazon](images/amzn_chart.png)

- High growth potential with noticeable volatility  
- Driven by expansion in AWS and innovation  
- Higher risk, higher reward profile  

---

## ⚙️ Technical Analysis
- Implemented **100-day moving average (100MA)** to track trends  
- Visualized:
  - Adjusted closing prices  
  - Volume and price movement  
- Used time-series analysis to compare stock behavior  

---

## 🧠 Key Insight

While all three companies showed long-term growth, Microsoft provided the best balance of:
- Stability  
- Consistent returns  
- Strong cloud market position  

---

## 🏆 Conclusion

Microsoft was identified as the **best investment option**, offering a combination of reliability, innovation, and sustained growth compared to Oracle and Amazon.
